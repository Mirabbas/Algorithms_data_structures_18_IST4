Задача: 
Реализовать головоломку Небоскребы.
В ней требуется заполнить числами квадрат так, чтобы в каждой строке и в каждом столбце каждое число использовалось лишь единожды. Заполнение нужно делать с помощью ключевых чисел, считая, что в каждой клетке внутри квадрата стоит дом с количеством этажей, равном стоящему в клетке числу, а числа вне поля означают, сколько домов видно в соответствующей строке или столбце, если смотреть от этого числа.

Необходимо:
- реализовать выбор размера игрового поля пользователем
- реализовать передачу введенных пользователем данных на обработку другому классу
- реализовать обработку исключений
- реализовать вывод решенной головоломки

Процесс:

 5.04.2020  + создан пример игрового поля
 6.04.2020  + принято решение,что цифры вне поля пользователь будет вводить сам
 7.04.2020  + разработан алгоритм заполнения поля, а также другие функции
 7.04.2020  + реализован вывод на экран решенной игры, а также учтены исключения (неправильный ввод)

Функции:
 HorizontalСheck - функция для проверки в значений в столбце
 main - функция, отвечающая за ввод данных и передачу их на обработку
 checking - функция проверки вводимых значений
 Fill - функция, отвечающая за заполнение поля
 PrintResult - функция,для вывода заполненного поля
 ValueSpace - функция, являющаяся вспомогательным методом для Print
