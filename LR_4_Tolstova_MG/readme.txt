Задача: 

Реализовать игру Небоскребы.

Небоскрёбы  – вид логической головоломки. В ней требуется заполнить числами квадрат так, чтобы в каждой строке и в каждом столбце каждое число использовалось лишь единожды. Заполнение нужно делать с помощью ключевых чисел, считая, что в каждой клетке внутри квадрата стоит дом с количеством этажей, равном стоящему в клетке числу, а числа внутри поля означают, сколько домов видно в соответствующей строке или столбце, если смотреть от этого числа.

Необходимо:

- реализовать создание карты игрового поля, где игроком будут расставляться цифры 
- реализовать создание игрового поля, куда будет входить карта и цифры-подсказки по краям
- реализовать проверку на правильность введенных данных
- реализовать вывод игрового поля на экран
- реализовать проверку на правильность прохождения игры


Процесс:

(16.04.2020) + реализация алгоритма проверки на правильность прохождения игры
(16.04.2020) + реализация вывода игрового поля на экран

Реализованные функции:

• main - запускает игру
• genereteMap - генерирует создание игровой карты
• StartGame - создает игровое поле, включая игровую карту и цифры-подсказки по краям
• changeRow - выбирает позицию по строкам
• changeColumn- выбирает позицию по столбцам
• setIndex - выбирает позицию цифры-подсказки
• setRow - выбирает позицию с левого и правого краев игрового поля
• setColumn - выбирает позицию с верхнего и нижнего краев игрового поля
• showMap- выводит игровое поле на экран
• show- выводит цифры по краям
• show- выводит игровую карту
• check- проверка на полную заполненность игрового поля
• check- проверка на сходство заполненной карты с правильной
